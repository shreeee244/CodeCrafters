# 🏎️ F1 MarketPrix Dashboard

A real-time Formula 1 analytics dashboard providing race insights, driver telemetry, and market intelligence for F1 enthusiasts and analysts.

## 🚀 Live Demo

**[View Live Dashboard →](https://code-crafters-beta.vercel.app/dashboard)**

## ✨ Features

- 📊 **Real-time Telemetry** - Live driver performance metrics
- 🏁 **Race Analytics** - Lap times, sector speeds, and tire degradation
- 📈 **Market Insights** - Team valuations and sponsor analytics
- 👤 **User Profile** - Personalized dashboard with saved preferences
- 🎨 **Modern UI** - Glass morphism design with smooth animations

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel
